# Manta
# 만타 프로젝트를 위한 시스템, 모터 제어 코드
------------------------------------------

#Description
만타 프로젝트는 관람객의 포즈를 인식하여 몸체의 색을 변화시키고 다양한 모션, 소리를 이용해 컨텐츠를 제공하는 프로젝트이다.

### 1. 기능

- trajectory를 이용한 날개 모션
- 단순한 색 변환이 아닌 hsv 영역에서 색을 변화하여 자연스러운 색 변화
- 간편한 turn on/off 시스템


### 2. 하드웨어

- rapberrypi 4
- dynamixel
- led strip
